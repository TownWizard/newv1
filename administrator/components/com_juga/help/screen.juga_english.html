JUGA - Overview

Close Window
JUGA is an add-on for the CMS Joomla! that allows you additional control over what content is available to which users.

* JUGA was authored by www.dioscouri.com
* The latest version is downloadable there
* Support is also there
JUGA consists of following menu items:
  • Config
    Adjust default JUGA settings
  • Groups
    Add, remove, and modify user groups
  • Site Items
    Modify JUGA settings for pages on your website
  • Assign Users
    Assign users to your groups
Toolbar Commands
  • Sync
  • Default
  • Withdraw
  • DefaultCE
  • Enroll/Withdraw
  • Sync
    • Synchronizes the JUGA database with your website's items.
Top
  • Default

    • This action adds the selected Items or Users to the default group, as defined in the Config Menu.
      Warning: You must have "default_juga" defined in the Config Menu for this to work properly.
Top
  • DefaultCE

    • This action sets the custom error URL = the default custom error URL, as defined in the Config Menu.
      Warning: You must have "default_ce" defined in the Config Menu for this to work properly.
Top
  • Withdraw
    • Removes selected Items and Users from *all* groups they are a member of.
Top
  • Enroll/Withdraw
    • Removes Items and Users from the selected groups.
